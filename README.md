# React test task

## Downloading

```bash
git clone https://github.com/demos-prog/react_test_task.git
```

## Starting the application

```bash
npm run dev
```

## Starting the application in docker

1. Run the command

```bash
npm run docker
```

2. navigate to `http://localhost:80` in browser

3. To stop the conatiner run:

```bash
docker-compose down
```
